# Inventory Management System

## 🚀 Overview
A full-stack inventory management application built using Spring Boot and Angular. This system allows users to manage inventory and process transactions in a scalable environment.

## 🛠 Tech Stack
- Java
- Spring Boot
- Angular
- MySQL
- Docker

## ⚙️ Features
- RESTful API for inventory operations
- Business rule validation
- Dynamic frontend with Angular
- Database integration

## 📦 Architecture
- Controller → Service → Repository pattern
- REST API communication

## ▶️ How to Run
1. Clone the repository
2. Run backend (Spring Boot)
3. Run frontend (Angular)

## 📌 Key Takeaways
- Built full-stack application
- Implemented REST APIs
- Containerized using Docker
